Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This struct holds information about the device available for Pcap
live capture.

A list of Pcap device descriptions is returned by
`io.createPcapDeviceDescList`.



.. rubric:: See also:

`io.createPcapDeviceDescList`
