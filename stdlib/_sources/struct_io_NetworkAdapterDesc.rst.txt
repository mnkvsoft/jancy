Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This struct holds information about the network adapter.

A list of network adapter descriptions is returned by
`io.createNetworkAdapterDescList`.



.. rubric:: See also:

`io.createNetworkAdapterDescList`
